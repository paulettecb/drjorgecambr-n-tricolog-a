# Assets del sitio

Estructura para cargar recursos reales del proyecto:

- `logo/` → Logo principal y favicon (`logo-dr-cambron.svg`, `favicon.svg`).
- `doctor/` → Foto principal del doctor (`dr-jorge-cambron.jpg` recomendado).
- `antes-y-despues/` → Resultados comparativos.
- `resultados-fue/` → Casos de injerto FUE.
- `educacion/` → Material educativo (posts, miniaturas, reels).
- `casos-reales/` → Evidencias de seguimiento clínico.

> Puedes reemplazar los `.gitkeep` por imágenes WebP/JPEG optimizadas.
